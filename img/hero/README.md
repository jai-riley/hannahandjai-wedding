# Hero Image

This folder contains the full-width background image shown at the top of the site.

## How to replace

1. Add your photo as `hero.jpg` (full resolution) and `hero-min.jpg` (compressed for web, ~200–400 KB).
2. The site uses `hero-min.jpg` — make sure that file exists.

**Recommended size:** 1920×1080px or wider, landscape orientation.
